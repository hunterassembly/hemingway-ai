# Code of Conduct

## Our Commitment

We want the Hemingway community to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Focus feedback on ideas and implementation, not people.
- Help newcomers with context when possible.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or hostile behavior.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove or edit comments, issues, or pull requests that violate this code of conduct.
Repeated or severe violations may result in temporary or permanent bans from project spaces.

## Reporting

To report behavior that violates this code, open a private communication channel through GitHub maintainers.

## Scope

This applies to all project spaces, including GitHub issues, pull requests, discussions, and related communication channels.
